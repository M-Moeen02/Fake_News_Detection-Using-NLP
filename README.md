# Fake News Detection System

Machine learning project comparing classical ML and deep learning for fake news detection.

## Models
- Logistic Regression
- Linear SVM
- BiLSTM (Deep Learning)

## Dataset
- 19,000+ news articles
- Balanced (50% fake, 50% real)
- Features: title, text, source, category
